Comment faire un mod Mindustry

Avant de commencer à coder

Pour fonctionner, un mod mindustry necessite:

Un fichier icon.png, pour l'icon du mod,

Un fichier mod.hjson pour les informations concernant le mod.

Ensuite, il faut créer un dossier nommé "content", dans lequel iront les scripts ainsi qu'un dossier "sprites" pour les images.

mod.hjson

Ce fichier doit contenir:

name: [Le nom du mod et du dossier]
displayName: [Le nom qui sera afficher]
description: [Une description du mod]
author: [Le créteur du mod]
version: [version du mod]
minGameVersion: [version du jeu]
